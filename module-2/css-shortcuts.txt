box-model: means a css element is a box of width 100% and height of 0%  by default. you can edit 
the properties as below

1. margin
2. padding 
3. border:
            border-width: defaults to medium
            border-style: defaults to None( so this have to be set)
            border-color: defaults to text color

            #  border shorthand
            border: 2px solid pink     [ border-width border-style border-color] 

            # applying a uniform border
            border: 2px solid pink     applies to all at once

            # applying a different border to the sides of an element
            p {
                border-top: 2px solid pink
                border-right: 5px dotted red
                border-bottom: 10px double green
                border-left: 1px dashed purple
            }
